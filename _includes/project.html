About the Project and the Course

Groktools is vinodkd's project submission for the Educational Technology course (CS6460), part of the Georgia Tech OMSCS

Groktools aims to fix gaps in program comprehension by building small tools that dovetail into existing dev workflows.

If you're interested to learn more about the project, a timeline is available, as are artifacts from the project

Georgia Tech's OMSCS program is the first online Masters from a top 5 Grad School

Educational Technology (CS6460) is an interesting, course that's run by Dr.David Joyner of Georgia Tech via the OMSCS program

Its a mini-PHD essentially, with an initial phase of intense literature study in one of 8-10 Ed Tech topics, followed by either a research or tool development project on a topic that you're narrowed down to. Some really awesome projects are happening there!
